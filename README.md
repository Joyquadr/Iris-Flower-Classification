# Iris-Flower-Classification

# Overview
This project classifies Iris flowers into three species using Machine Learning.

# Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

# Models Used
- Decision Tree
- Logistic Regression
- K-Nearest Neighbors (KNN)

# Dataset
Scikit-learn Iris Dataset

# Results
Compared three classification algorithms and evaluated their performance using accuracy and confusion matrices.

# Skills Demonstrated
- Data preprocessing
- Exploratory Data Analysis
- Model training
- Model evaluation
- Classification
